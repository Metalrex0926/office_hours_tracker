# PocketBase Import Package

This folder is for one-click schema import in new environments.

## Goal
Avoid manual collection setup every time.

## Maintainer Workflow (one-time or when schema changes)
1. Configure collections in PocketBase dashboard (`users`, `user_state`, rules).
2. Go to `Settings -> Export collections`.
3. Save/export file as:
   - `pocketbase/pb_schema.json`
4. Commit and push the updated `pb_schema.json`.

## New User Workflow (clone and start)
1. Start PocketBase and create first superuser.
2. Open dashboard: `http://127.0.0.1:8090/_/`.
3. Go to `Settings -> Import collections`.
4. Click `Load from JSON file` and choose:
   - `pocketbase/pb_schema.json`
5. Click `Review` -> `Import`.
6. Set app `.env` and run Expo app.

## Notes
- `pb_schema.json` is intentionally not generated by code; it is exported from your real PocketBase config.
- For physical phone testing, use computer LAN IP in `EXPO_PUBLIC_POCKETBASE_URL`.
