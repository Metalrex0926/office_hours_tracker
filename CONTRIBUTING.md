# Contributing

Thanks for contributing.

## Local Setup
1. Use Node 24
   ```bash
   nvm install 24
   nvm use 24
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Run tests
   ```bash
   npm test
   ```

## Pull Requests
- Keep PRs focused and small.
- Add or update tests when behavior changes.
- Update docs (`README.md`) if setup or behavior changes.
- Ensure CI is passing before requesting review.

## Commit Style
Use clear commit messages in imperative form, e.g.:
- `Fix sync merge on stale remote payload`
- `Add validation tests for leave date range`
